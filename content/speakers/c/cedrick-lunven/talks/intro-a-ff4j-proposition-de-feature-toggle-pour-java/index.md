  title: Intro à FF4J, Proposition de Feature Toggle pour Java
  abstract: Le feature toggle, ou la capacité à activer ou désactiver des fonctionnalités
    à chaud, est souvent associée à une astuce de développement au service du Continuous
    Delivery. On peut pousser du code en production qui n'est pas complètement finalisé
    et introduire de la dette technique... mais il peut être tellement plus utile
    !!!!! Après quelques cas d'usages très concrets, je vous propose de détailler
    les capacités du framework FF4J, les choix d'implémentation, les références en
    production, les killers features ainsi que les prochaines évolutions.
