  title: Dart, le nouveau langage de Google pour le web
  abstract: Dans la dernière Devoxx, Google a annoncé la sortie de la version 1.0
    de leur nouveau langage pour le web, Dart. Derrière cette v1 il y a deux ans de
    développement dans lesquels des développeurs de tout le monde ont pu collaborer
    pour essayer de créer un langage structuré, facile d'apprentissage pour tout développeur
    quel que soit son background et surtout adapté au web d'aujourd'hui. Prenant les
    meilleures idées de Java, JavaScript et C#, Dart se veut ce que JavaScript aurait
    pu être s'il avait été développé aujourd'hui.
