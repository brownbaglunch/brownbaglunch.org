  title: Spring Boot & Containers - Do's & Don'ts
  abstract: Bien que Docker soit facile d'utilisation, construire une image reste
    un exercice compliqué. Optimisation, layers, et sécurité nécessitent de s'abstraire
    des Dockerfile basiques qu'on peut trouver sur internet. Dans ce BBL, nous verrons
    comment bien packager une application Spring Boot dans une image Docker/OCI. Nous
    allons parcourir les bonnes pratiques préconisées par Spring, les pièges à éviter,
    comment bien écrire un Dockerfile pour optimiser la construction et l'image finale
    pour une exécution dans Kubernetes et nous verrons aussi des moyens alternatifs
    de construction d'images Spring Boot et l'outillage d'analyse à ajouter à votre
    toolbox.
