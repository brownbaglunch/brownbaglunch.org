  title: La valeur du bon code pour les non-devs
  abstract: La valeur métier du bon code est difficile à appréhender, même parfois
    pour les développeurs. Comment pouvons-nous espérer qu'on nous demande d'en faire?
    Dans ce démo je rend compréhensible aux non techniques la valeur que leur apporte
    un code en bon état. Dans ce démo il parait aussi comme une évidence que pour
    passer d'un mauvaise état à un bon état il n'est pas tellement une question d'allouer
    du temps sur le moment, mais d'avoir de l'entrainement.
