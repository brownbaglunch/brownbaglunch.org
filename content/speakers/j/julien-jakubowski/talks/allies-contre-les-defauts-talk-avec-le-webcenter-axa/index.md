  title: Alliés contre les défauts (talk avec le WebCenter AXA)
  abstract: La pratique de la revue de code prévient énormément de défauts. Mais elle
    a aussi d’autres “effets kiss-cool”. C’est une pratique qui existe depuis une
    soixantaine d’années. Mais pourquoi est-elle si rarement rencontrée dans les DSI
    ? Sans doute parce qu’elle est méconnue, mais aussi parce qu’elle fait l’objet
    d’appréhensions. Faites-vous des revues de code chez vous ? Et sinon, pourquoi
    pas ? Peut-être pensez-vous que cela prend trop de temps, que c’est trop cher,
    qu’en jugeant le code on jugera l’auteur… Au Webcenter AXA, le code est revu systématiquement
    avant de partir en production. Cela porte ses fruits. Mais cela ne s’est pas fait
    sans accrocs. Venez découvrir un retour d’expérience.
